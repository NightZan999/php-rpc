# Xolentum PHP-RPC

This are the RPC scripts in PHP for querying `xolentumd`

## Configuration

Set the daemon host and the **daemon RPC** port in `lib/config.php`. 

The default daemon RPC ports are:

* Mainnet  - 13580
* Testnet  - 44442
* Stagenet - 33332
